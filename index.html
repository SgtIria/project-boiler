Introduction

This is a simple learning project to create a full stack skeleton project which covered my domain skill set. It really about helping me understand:

NPM / Node

  1. Multiple build paths (Dev/Prod)
  2. NPM Scripts (rather than overweight build tools)

Effective workflow paths

  1. Managing the flow of changes within the project into destination builds
  2. Live reload and continence set-up (we all expect this now)
  3. Capturing errors and the resolution cycle
  4. Testing

Pre Processors (Who need to type more than we already have to)

  1. CofeeScript Im still using coffeescript (it works and ES6 feel clumsy atm)
  2. Stylus
  3. Jade

Libraries

  1. Riot JS
  2. Q (Promises)
  3. LoDash (I just like functional style atm)
  4. Jeet (Grid layout assistance)
  5. Kyoto (Style mixins)

Git

  1. Proficiency with git and github
  2. Familiarity with github pages system

Vim

1 Supportive editor environment, pathogen setup et al.
